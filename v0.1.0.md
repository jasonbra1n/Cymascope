# Cymascope v0.1.0 Release Notes

This is a significant feature release, transforming the initial proof-of-concept into a full-featured, interactive web application. This version introduces a host of new capabilities, a complete UI overhaul, and foundational improvements for future development.

## âœ¨ New Features

- **Tuner Overlay**: A real-time tuner has been added, displaying the dominant incoming frequency and its closest musical note.
- **Selectable Tuning Systems**: Users can now switch the tuner's reference between the standard A=440Hz and the alternative A=432Hz.
- **Frequency Scaling**: A new "Frequency Scale" slider gives users direct control over how audio frequencies are mapped to the visual patterns.
- **Theme Switching**: The application now supports both Light and Dark themes, with a toggle to switch between them. The user's preference is saved locally.
- **Theme Sync for Embedding**: The app can now listen for theme changes from a parent application (like the LAB Digital Workshop) and update its theme accordingly.

- **Adaptive UI**: All UI elements, including the new overlays, are now theme-aware, ensuring excellent readability and a consistent look in both light and dark modes.
- **Compact Header**: The header has been redesigned to be more compact and navbar-like, giving more vertical space to the main content.
